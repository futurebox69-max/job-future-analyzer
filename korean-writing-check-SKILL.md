## 2. 번역투 점검 — 영어 직역 구문 제거

> 출처: 이영옥 2001, 김도훈 2009, 김정우 2007, 김혜영 2019, 박옥수 2018, im-not-ai v2.0

영어·일본어식 구문을 한국어로 무리하게 옮긴 흔적. **AI 글의 가장 결정적 시그니처.**

### 2-1. S1 결정적 — 한 번만 나와도 AI 티 (무조건 제거)

```
□ A-1  "~에 대해(서)" 남발
       → 목적격 조사로 직결 ("AI 규제에 대해 논의" → "AI 규제를 논의")

□ A-2  "~를 통해/통하여" 남발
       → "~로", "~해서", "~함으로써"로 분산

□ A-3  "~에 있어(서)"
       → "~에서", "~을 볼 때"

□ A-7  "가지고 있다" / have·make·take·give + 명사 직역
       → 형용사·동사 환원 ("경쟁력을 가지고 있다" → "경쟁력이 강하다")
       → "회의를 가지다" → "회의를 했다"

□ A-8  이중 피동 "~되어진다" / "~지게 된다"
       → 능동 또는 단일 피동 ("판단되어진다" → "판단된다")

□ A-16 "그/그녀/그것/그들" 대명사 직역 (한 단락 3회+)
       → 50%+ 주어 생략(영형 대명사) 또는 호칭·명사구로
       → 소유격 "그의 손" → "손" (거의 항상 잉여)
```

하나라도 발견되면 **미통과**.

### 2-2. S2 강함 — 3회 이상 반복 시 AI 티

```
□ A-4  "~라는 점에서" (3회+)
       → "~서", "~라는 이유로"

□ A-5  "~와 관련하여/관련된"
       → "~에", "~의"

□ A-6  "~에 기반하여/바탕으로" 남발
       → "~로", "~을 보고"

□ A-9  "~에 의해" 피동 (by-passive 직역)
       → 행위자를 주어로 ("AI에 의해 생성" → "AI가 만든")

□ A-10 "~할 수 있다" 남발 (가능형 서술)
       → 확정 서술 ("높일 수 있다" → "높인다")

□ A-11 "~을 위해" 목적절 남발
       → "~려고", "~도록"

□ A-12 "만들어지다" / "이루어지다" 자동 피동
       → "합의가 이루어졌다" → "합의했다"

□ A-13 명사 나열 (조사 생략, 영어식 명사구)
       → 조사 "의/가/를" 복원

□ A-15 추상 주어 + 만능 동사
       → "기술이 보여준다/제공한다/가져온다" → 구체 주어로 환원
       → 사역 타동사: "X 때문에/덕분에" 부사절로
       → 인지 동사(suggest/show): "~에 따르면 ~이다"로 분리

□ A-18 관계절 좌향 수식 (명사 앞 3어절+ 관형구)
       → 문장 분리 또는 후치 동격절

□ A-19 이중 조사 결합 (~에서의/~에로의/~으로의/~에의/~으로부터의)
       → 절·구로 풀어쓰기 (단순 "~의"는 비대상)
```

한 문서에 S2 패턴 합산 5회+ 발견 시 **미통과**.

## 3. AI 글투 점검 — 구조·관용구·리듬·수식·접속사

### 3-1. 구조적 AI 패턴

```
□ C-1  기계적 병렬 열거 "첫째·둘째·셋째"
       → 서술문으로 녹이거나 어휘 변주

□ C-5  이모지 남발 (✅🚀💡⚠️📊)
       → 에세이·리포트면 전량 제거 [S1]

□ C-7  "먼저·반면·결국" 3단 공식
       → 3개 중 2개 삭제

□ C-9  숫자 괄호 인덱싱 "1) 2) 3)"
       → 본문에 녹이기

□ C-10 콜론 부제 헤딩 "X: Y" 반복
       → 단일 명사구·동사구로 압축 [S1]

□ C-11 연결어미 뒤 쉼표 [S1 — 단일 최강 탐지 지표]
       → "-고," "-며," "-지만," "-면서," 직후 쉼표 제거
       → 인간 4% vs AI 20% (KatFish, 4.84배 분리도)
       → 6회+ = 강한 AI 신호

□ C-12 쉼표 포함률 50%+ (문서 단위)
       → 일부 문장의 쉼표를 마침표 분할 또는 연결어미로 흡수
```

C-5, C-10, C-11 중 하나라도 발견 시 **미통과**.

### 3-2. AI 특유의 관용구 (Signature Phrases) — 발견 즉시 교체

```
□ D-1  결산 lexicon: "결론적으로/따라서/이를 통해/그러므로/요약하면/정리하면"
       → 3회 초과 시 삭제 또는 치환 [S1]

□ D-2  의의 과장: "시사하는 바가 크다/주목할 만하다/간과할 수 없다"
       → 삭제 또는 구체 결론으로 [S1]

□ D-3  열거 도입: "크게 세 가지로 나눌 수 있다/다음과 같은 특징을 가진다"
       → 삭제

□ D-4  hype 어휘: "혁신적/획기적/전례 없는/압도적/파격적/폭발적"
       → 구체 수치·사실로 환원 [S1]

□ D-5  의인화 추상 주어: "기술이 묻는다/시대가 부른다/변화가 요구한다"
       → 사람·기관 주어로 [S1]

□ D-6  결말 공식: "~할 때입니다/~해야 할 시점입니다/지금이야말로"
       → 평서로 닫기 [S1]

□ D-7  변환 공식: "X에서 Y로" 반복
       → 한 문서에 1회만
```

D-1~D-6 중 하나라도 발견 시 **미통과**.

### 3-3. 리듬·문장 길이

```
□ E-1  문장 길이 균일 (모든 문장 30~50자 부근)
       → 단문(10~15자) + 장문(80자+) 의도적 혼용

□ E-2  동일 종결어미 4문장 연속 ("~다. ~다. ~다. ~다.")
       → "~었다·~ㄴ다·~기 마련이다·~ㄹ 것이다" 다양화
       → 진행형 "~고 있다" 자동 매핑 → 단순 시제 환원 가능 시 환원

□ E-3  모든 문단 3~4문장 공식
       → 1문장 문단 / 6문장 문단 의도적 혼용

□ E-4  단문 일변도 (복문·중문 부재)
       → 연결어미(-며·-고·-는데·-면서)로 묶어 복문화
       → 단문 비중 60% 전후, 복문·중문 30% 이상
```

E-2(4문장 연속 동일 종결어미) 발견 시 **미통과**.

### 3-4. 과도한 수식·중복

```
□ F-1  정도부사 중독: "매우/정말/진짜로/대단히/극히"
       → 대부분 삭제. 강조 필요 시 구체 수치·사례

□ F-2  동의어 이중 수식: "중요하고 핵심적인/새롭고 혁신적인"
       → 하나만 남김

□ F-4  한자어 명사화 "-성/-적/-화" 누적 (12회+)
       → 동사·형용사 어근으로 환원

□ F-5  "~적 N" 추상 체인 (전략적 함의·구조적 변화·실천적 기반)
       → "전략 함의", "구조가 바뀐다" 식으로 해체
       → 문서 전체 "~적 N" 밀도 절반 이하로
```

### 3-5. Hedging (과도한 완곡)

```
□ G-1  추측형 종결 남발: "~할 수 있을 것으로 보인다/~인 것으로 판단된다"
       → 단언할 수 있는 곳은 단언

□ G-2  이중·삼중 완곡: "~할 가능성이 있을 수 있다"
       → 하나만 남김

□ G-3  안전 균형 lexicon: "양쪽 모두/두 가지 모두/장점도 있지만/신중하게/균형"
       → 4회 초과 시 화자 입장으로 치환
```

### 3-6. 접속사 남발

```
□ H-1  문두 접속사 과다: "또한/따라서/즉/나아가/아울러/게다가/더욱이"
       → 70% 이상 제거 [S1]

□ H-3  메타 진입 반복: "이는~/이 점에서~/이 관점에서~/이 말은~"
       → 본문에 녹이거나 삭제

□ H-4  "즉" 남발
       → 한 문서에 2회 이하
```

H-1 (5회+) 발견 시 **미통과**.

### 3-7. 형식명사·의존명사 과다

```
□ I-1  "~인 것이다/~한 것이다" 종결 남발
       → 평서형 "~다"로 [S1]

□ I-3  "~다는 뜻이다/~다는 의미다" 결말
       → "~다" 직접 종결

□ I-4  권고형 결말 "~해야 한다/~해야 합니다" 반복 (5회+)
       → 구체 동사 단언

□ I-6  "~능력" 추상명사 연쇄 (3회+)
       → 동사형 풀기 ("사고 능력" → "잘 사고한다")
```

### 3-8. 시각 장식 남용

```
□ J-2  따옴표 강조 5회+
       → 핵심 1~2개만 [S1]

□ J-3  대시(—) 남용
       → 1문서 1~2회 이하

□ J-4  괄호 부연 과다: "(이는 ~을 의미한다)"
       → 본문화 또는 삭제
```

## 4. 글쓰기 품질 점검

### 4-1. 금지 어휘 점검

```
□ 연결사: "또한·더불어·뿐만 아니라·나아가·아울러"
□ 나열: "첫째·둘째·셋째" 식 도식 나열
□ 과잉 수식: "매우·정말로·확실히·상당히·굉장히"
□ 모범답안 도입: "오늘은 ~에 대해 알아보겠습니다"
□ 정의문 도입: "X는 Y입니다"
□ "필자는" → "나는" 또는 주어 생략
□ 이모지·ㅋㅋ·ㅎㅎ
```

하나라도 발견 시 **미통과**.

### 4-2. 추상어 → 장면 점검 (★ 가장 중요)

추상어 목록 — 문제, 구조, 불안, 변화, 중요, 본질, 위기, 회복, 소통, 사명, 부르심, 사역, 영성, 신앙, 공동체, 교회, 사회, 시대, 기술

```
□ 추상어 뒤에 장면이 내려왔는가?
  장면 = 시간·장소·행동·표정·몸의 반응·실제 들릴 만한 말
□ 500자 단위로 장면이 한 번도 안 나오는 구간이 있는가?
□ 추상어만 떠다니고 한 사람의 모습이 안 보이는가?
```

추상어 뒤 장면 없으면 **미통과**.

### 4-3. 정보·해석·연결·판단 점검

```
□ 정보(무슨 일)만 있는 단락이 연속되는가?
□ 해석(무엇을 뜻하는가)이 빠졌는가?
□ 판단(글쓴이의 시선)이 보이지 않는가?
```

모든 단락이 4개 중 최소 2개. 정보만 있는 단락 연속 시 **미통과**.

### 4-4. 도입·마무리 점검

**도입**
```
□ 첫 문장이 팩트·장면·질문 중 하나인가?
□ "오늘은~", 정의문 도입이 아닌가?
```

**마무리**
```
□ 결론이 매끈하게 봉합되어 있는가?
□ 좋은 말로만 끝내려 했는가?
□ 행동·질문·다음 단계의 가능성이 남아 있는가?
```

매끈한 봉합이면 **미통과**.

### 4-5. AI 글 티 종합 점검

```
□ 모든 문장이 너무 반듯한가?
□ 너무 친절하게 하나부터 열까지 설명하는가?
□ 뻔히 아는 말을 반복하는가?
□ 추상어가 많은데 장면이 없는가?
□ 판단 없이 균형만 잡는 척하는가?
□ 같은 뜻을 다른 말로 두 번 썼는가?
□ "~에 대해서·~에 있어서·~라는 측면에서" 군더더기가 있는가?
□ 연결어미 뒤 불필요한 쉼표가 있는가?
```

8개 중 3개+ 해당 시 **미통과**.

## 5. 책 전용 점검 — book-master-os와 함께 호출 시에만

### 5-1. 사고 OS 점검

```
□ 통찰력(이해→분석→예측), 시스템사고, 행동과학, TRIZ 중 최소 3축?
□ 신학적 진술이 실천 가능한 행동으로 내려왔는가?
```

### 5-2. 합동 신학 표준 점검

```
□ 합동(총신대) 체계 안에 있는가?
□ 사명 ≠ 직업 (직업은 그릇)
□ 사명 ≠ 자아실현
□ 사명 ≠ 신비주의
□ 부르심 → 사명 순서
□ 어휘(사명·소명·부르심·사역·천직) 일관성
□ "필자는" 0건
```

### 5-3. 직업장(9~14장) 6단계 흐름 점검

```
1. □ 장면(현재)
2. □ 변화 — AI가 무엇을 가져가는가
3. □ 새로운 결핍 — AI가 못 채우는 자리
4. □ 정의의 재구성
5. □ 신학적 해석
6. □ 교회와 개인의 준비 — 구체적 행동 1~2가지
```

6단계 중 하나라도 빠지면 **미통과**.

## 7. 자주 걸리는 Top 8

1. **연결어미 뒤 쉼표** — AI 최강 시그니처. "-고," "-며," "-지만," 쉼표 전량 제거.
2. **추상어 뒤 장면 부재** — "변화", "본질", "구조" 뒤에 한 사람의 모습이 없으면 AI 글.
3. **"~에 대해서/~를 통해/~에 있어서"** — 번역투 3대장. 목적격 조사로 직결.
4. **결론 매끈한 봉합** — "우리는 함께 ~해야 한다" 식 마무리. 질문이나 열린 결말로.
5. **"~할 수 있다" 남발** — 가능형을 남발하면 글이 약해진다. 단언으로.
6. **정보만 있고 판단 없음** — 균형만 잡힌 글. 글쓴이 시선이 보여야 한다.
7. **어휘 흔들림** — 사명·소명·부르심·사역 섞어 쓰기 (책 전용).
8. **hype 어휘** — "혁신적/획기적/전례 없는" → 구체 사실로.

## 8. 이 스킬의 경계

이 스킬은 다음을 하지 않는다.

- **글을 직접 다시 쓰지 않는다.** 점검 결과와 수정 방향만 보고.
- **사실 검증(팩트 체크)은 별도.** 글쓰기 형식과 번역투에 한정.
- **신학 자체를 최종 판정하지 않는다.** 1차 점검만.
- **변경률 50% 초과 수정은 금지.** 과윤문 방지.
