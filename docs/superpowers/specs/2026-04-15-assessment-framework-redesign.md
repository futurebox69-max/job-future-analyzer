# BTS 분석 — 미래역량 검사 프레임워크 전면 재설계 스펙

## 서비스 구조

| 서비스명 | 역할 | 비고 |
|---------|------|------|
| **BTS** | 본진 (메인 사이트) | — |
| **BTS Forecast** | 내 직업의 미래 분석 | job-future-analyzer.vercel.app |
| **BTS 분석** | 미래역량 검사 전문 사이트 | **이 스펙의 대상** |

BTS Forecast와 BTS 분석 모두 **로그인 필수**. 이 스펙은 **BTS 분석** 사이트의 검사 프레임워크를 다룬다.

## 목표

BTS 분석의 측정 프레임워크, 검사 방법, 결과 프레젠테이션을 근본부터 재설계한다. 기존 6차원·단일검사 체계를 7차원·4검사 체계로 전환하고, 프로필 기반 맞춤 시나리오와 게임화된 문항 형식으로 몰입도와 신뢰도를 동시에 확보한다.

## 설계 원칙

1. **정확성 우선** — 검사 결과가 정확해야 한다. 재미는 정확성을 해치지 않는 범위 내에서.
2. **1회 완결 + 누적 정밀화** — 1개 검사만으로도 의미 있는 결과, 더 할수록 정확해지는 구조.
3. **내 이야기로 검사** — 프로필 기반 맞춤 시나리오로 "내 상황이 아닌" 문제 제거.
4. **검사가 아닌 게임** — 지루하지 않고 쉽고 재미있는 경험.

## 서브프로젝트 구성

이 스펙은 3개의 서브프로젝트로 나뉘며, 순서대로 설계·구현한다:

| 서브프로젝트 | 범위 | 의존성 |
|------------|------|--------|
| **A. 검사 프레임워크** | 7차원 정의, 유형 분류, 점수 체계 | 없음 |
| **B. 검사 방법·문항** | 4개 검사, 테마팩, 프로필 맞춤, 게임화 | A 완료 후 |
| **C. 결과 프레젠테이션** | 단계적 공개, 누적 결과, AI 심층 리포트 | A, B 완료 후 |

이 문서는 **전체 스펙**을 포함한다. 구현 플랜은 서브프로젝트별로 별도 작성한다.

## A. 검사 프레임워크

### 7차원 미래역량 모델

기존 6차원(구조적사고, 창의적재설계, 감성연결, 적응민첩성, 윤리적판단, 협업지능)을 폐기하고, WEF·OECD·UNESCO·P21·DigComp 프레임워크 교차 분석 결과에 기반한 7차원 모델로 전환한다.

| # | 차원 키 | 아이콘 | 한글명 | 영문명 | 설명 | 기존 대비 |
|---|--------|--------|--------|--------|------|----------|
| 1 | `insight` | ⭐ | 통찰력 | Insight | 이해→분석→예측의 3단계 사고력 | 구조적사고 흡수 + 예측력 추가 |
| 2 | `creative` | 🎨 | 창의적 혁신 | Creative Innovation | 기존 틀을 깨고 새로운 가치를 만드는 능력 | 유지·강화 |
| 3 | `ai_fluency` | 🤖 | AI 활용 지능 | AI Fluency | AI 도구를 이해하고 효과적으로 활용·판단하는 능력 | **신규** |
| 4 | `emotional` | 💎 | 감성·소통 지능 | Emotional Intelligence | 공감, 신뢰 구축, 설득과 영향력 | 감성연결 확장 |
| 5 | `adaptive` | 🔄 | 적응 회복력 | Adaptive Resilience | 변화에 빠르게 적응하고 실패에서 회복하는 능력 | 적응민첩성 + 회복탄력성 |
| 6 | `ethical` | ⚖️ | 윤리적 판단력 | Ethical Judgment | AI 시대의 가치 판단과 책임 있는 의사결정 | 유지 |
| 7 | `collab` | 🤝 | 협업 리더십 | Collaborative Leadership | 사람+AI와 함께 목표를 달성하는 능력 | AI협업 확장 |

### 통찰력 하위 점수

통찰력은 핵심 역량으로, 3개 하위 점수를 별도로 측정·표시한다:

| 하위 점수 | 키 | 설명 |
|----------|-----|------|
| 이해력 | `insight_understand` | 복잡한 정보를 빠르게 파악하는 능력 |
| 분석력 | `insight_analyze` | 패턴과 핵심을 찾아내는 능력 |
| 예측력 | `insight_foresight` | 미래 변화를 예상하고 대비하는 능력 |

통찰력 종합 점수 = 3개 하위 점수의 균등 평균 (각 1/3 가중치).

### 유형 분류 체계

7차원 점수 조합에 따라 규칙 기반(rule-based)으로 유형을 분류한다. 기존 10개 유형을 12개로 확장한다.

**분류 알고리즘 (우선순위 순):**

1. 모든 차원 ≥ 75, 최저 ≥ 60 → ⭐ **올라운더**
2. 통찰력 ≥ 85 + AI활용 ≥ 80 → 🔮 **미래 설계자** (통찰 특별 유형)
3. 통찰력 ≥ 85 → 🧠 **통찰 전략가** (통찰 특별 유형)
4. 최고-최저 차이 ≤ 10 → 🎯 **균형 전략가**
5. 최저 ≤ 35, 최고 ≥ 90 → 🌋 **잠재력 폭발형**
6. 상위 2개 차원 차이 ≤ 5 → ⚔️ **듀얼 스페셜리스트**
7. 나머지 → 최고 차원에 따라 단일 전문가 유형:
   - `insight` → 🧠 통찰 전략가
   - `creative` → 🎨 창조적 파괴자
   - `ai_fluency` → 🤖 AI 네이티브
   - `emotional` → 💎 공감 리더
   - `adaptive` → 🔄 적응형 혁신가
   - `ethical` → ⚖️ 윤리 수호자
   - `collab` → 🤝 시너지 메이커

총 12개 유형: 올라운더, 미래설계자, 통찰전략가, 균형전략가, 잠재력폭발형, 듀얼스페셜리스트, 창조적파괴자, AI네이티브, 공감리더, 적응형혁신가, 윤리수호자, 시너지메이커.

### 점수 체계

- 각 차원: 0~100 정규화 점수
- 통찰력 하위 3개: 각 0~100 (통찰력 검사에서만 세분화 측정, 다른 검사에서는 통찰력 종합만 측정)
- 종합 점수: 7차원 가중 평균 — 통찰력 20%, 나머지 6개 각 13.3%
- 신뢰도 점수: 완료한 검사 수에 따른 표시 등급 (아래 참조)

### 신뢰도 등급

| 완료 수 | 등급 | 표시 | 의미 |
|---------|------|------|------|
| 1개 | ⭐ 탐색 | "탐색 수준" | 방향성 참고용 |
| 2개 | ⭐⭐ 참고 | "참고 수준" | 경향성 파악 가능 |
| 3개 | ⭐⭐⭐ 신뢰 | "신뢰 수준" | AI 심층 리포트 해금 |
| 4개 | ⭐⭐⭐⭐ 정밀 | "정밀 수준" | 풀 프로필 달성 |

이 등급은 통계적 신뢰구간이 아닌 **UX 표시 등급**이다. 실제 정밀도는 검사 수·문항 수·응답 일관성에 따라 달라진다.

## B. 검사 방법·문항

### 4개 검사 체계

사용자는 4개 검사를 독립적으로, 원하는 순서로, 원하는 시간에 수행한다. 로그인 기반이므로 진행 상황이 저장된다.

| # | 검사명 | 포함 문항 형식 | 문항 수 | 시간 |
|---|--------|--------------|--------|------|
| 1 | 🎭 시나리오 검사 | 시나리오 선택 4 + 갈등 중재 2 | 6문항 | ~5분 |
| 2 | ⚡ 직감 검사 | 순간 반응 6 + 우선순위 배열 2 | 8문항 | ~4분 |
| 3 | 🧩 통찰력 검사 | 패턴 발견 3 + 미래 예측 3 | 6문항 | ~5분 |
| 4 | 🤖 AI 활용 검사 | AI 판단 3 + 실용 시나리오 3 | 6문항 | ~4분 |

### 검사-차원 커버리지 매트릭스

각 검사가 어떤 차원을 측정하는지 (●=주력, ○=부분):

| 차원 | 🎭 시나리오 | ⚡ 직감 | 🧩 통찰력 | 🤖 AI활용 | 측정 횟수 |
|------|-----------|--------|----------|----------|----------|
| 통찰력 (종합) | ○ | ○ | ● | ○ | 4회 |
| 창의적 혁신 | ○ | ● | ○ | | 3회 |
| AI 활용 지능 | | | | ● | 1회 |
| 감성·소통 지능 | ● | ○ | | | 2회 |
| 적응 회복력 | | ● | ○ | ○ | 3회 |
| 윤리적 판단력 | ● | | | ● | 2회 |
| 협업 리더십 | ● | ○ | | | 2회 |

**통찰력 하위 점수 측정:** 통찰력 검사(3번)에서만 이해·분석·예측 하위 점수를 개별 측정한다. 다른 검사에서는 통찰력 종합 점수만 측정한다.

**각 검사는 7차원을 부분적으로 측정**하므로 1개만 해도 기본 결과를 제공할 수 있다. 3개 이상 하면 대부분의 차원이 2회 이상 교차 측정되어 높은 신뢰도를 확보한다. AI활용지능은 검사 4에서만 측정하므로, AI활용 점수가 필요하면 검사 4를 반드시 수행해야 한다.

### 누적 점수 통합 알고리즘

여러 검사 결과를 하나의 7차원 점수로 통합하는 방법:

1. **단일 검사**: 해당 검사에서 측정된 차원만 점수 표시. 미측정 차원은 "미측정"으로 표시.
2. **다중 검사 — 가중 평균**: 같은 차원을 여러 검사에서 측정한 경우, 각 검사의 해당 차원 점수를 **문항 수 가중 평균**으로 통합.
   - 예: `ethical`이 시나리오 검사(2문항, 72점)와 AI활용 검사(2문항, 68점)에서 측정 → 통합 점수 = (72×2 + 68×2) / (2+2) = 70점
3. **주력(●) vs 부분(○) 가중치**: 주력 측정은 가중치 1.0, 부분 측정은 0.6. 이를 문항 수에 곱하여 가중 평균 계산.
4. **미측정 차원 처리**: 어떤 검사도 측정하지 않은 차원은 결과에서 "이 차원은 아직 측정되지 않았습니다. [해당 검사]를 해보세요"로 안내.

### 문항 형식 상세

#### 1. 시나리오 선택
- 상황 제시(2줄 이내) + 4지 선택
- 선택마다 즉각 피드백 ("오, 대담한 선택!")
- 드라마/웹툰 느낌의 스토리 전개

#### 2. 순간 반응 (3초 제한)
- 2개 선택지, 3초 타이머
- 틱톡 스와이프 느낌의 빠른 템포
- 직감적 성향 측정 (의식적 판단 배제)

#### 3. 우선순위 배열
- 5~6개 항목 드래그 정렬
- 전략적 사고와 가치 체계 측정

#### 4. 패턴 발견
- 데이터/이미지에서 규칙이나 이상점 찾기
- 퍼즐/퀴즈쇼 느낌
- 통찰력(분석) 직접 측정

#### 5. 미래 예측
- "3년 후 어떻게 될까?" 시나리오
- 여러 미래 시나리오 중 선택 + 근거 선택
- 통찰력(예측) 직접 측정

#### 6. AI 판단
- 실제 AI 결과물(텍스트/이미지)을 보여주고 오류/한계 찾기
- 탐정 놀이 느낌
- AI활용지능 직접 측정

#### 7. 갈등 중재
- 대립하는 두 입장 제시 → 해결책 선택
- 감성·윤리·협업 교차 측정

### 프로필 기반 맞춤 시나리오

#### 프로필 수집 항목
검사 시작 전 수집:
- **성별**: 남/여/기타
- **연령대**: 10대/20대/30대/40대+
- **직업군**: 학생/직장인/자영업/구직중/주부/기타
- **(선택) 세부 분야**: IT/교육/의료/서비스/제조/금융/기타

#### 테마팩 시스템

프로필 기반으로 **상위 3개 테마팩을 추천**(첫 번째가 디폴트)하고, **사용자가 최종 선택**한다. 추천은 소프트 서제스천이며, 사용자는 어떤 테마팩이든 자유롭게 선택 가능하다.

| 테마팩 | 기본 추천 대상 | 시나리오 분위기 |
|--------|--------------|---------------|
| 🎮 게임 월드 | 10~40대 남성 (기본 추천) | LoL, 마크, 로블록스, 발로란트 상황 |
| 💄 라이프스타일 | 10~30대 여성 (기본 추천) | 뷰티, SNS, 트렌드 상황 |
| 💼 직장 서바이벌 | 30대+ 직장인 | AI도입, 이직, 팀갈등 상황 |
| 🏪 사장님 모드 | 자영업/창업 | 매출, 직원관리, AI활용 상황 |
| 📚 진로 탐색 | 중고등학생 | 학교, 친구, 진로고민 상황 |
| 🎓 취업 전쟁 | 대학생/취준생 | 면접, 스펙, 경쟁 상황 |
| 👨‍👩‍👧 부모 모드 | 학부모 | 자녀교육, 가정, AI시대 육아 |

**추천 로직:** 프로필(성별+연령대+직업군) 조합으로 3개를 추천. 예: 20대 여성 직장인 → [라이프스타일, 직장 서바이벌, 게임 월드]. 사용자가 다른 테마를 선택해도 무방하며, 테마 변경은 결과에 영향을 주지 않는다 (측정 구조 동일).

#### 맞춤화 원칙

**문항의 심리측정 구조(어떤 선택이 어떤 역량에 점수를 주는지)는 테마팩 간 동일하다.** 포장(시나리오 텍스트, 이모지, 등장인물, 배경)만 테마에 맞게 변경한다. 이를 통해:
- 결과는 테마팩에 관계없이 동일한 7차원 척도로 비교 가능
- 사용자는 "내 세계관"의 이야기로 몰입

#### 남성 게임 시나리오 세부 매핑

남성 사용자, 특히 10~40대 남성에게는 유명 게임 상황을 적극 활용한다:

| 연령대 | 주요 게임 | 시나리오 예시 |
|--------|----------|-------------|
| 10대 | 로블록스, 마크, 발로란트 | "마크에서 친구랑 만든 서버가 그리퍼 공격을 받았습니다" |
| 20대 | LoL, 오버워치, 배그 | "랭크에서 트롤이 나타났는데 나만 캐리할 수 있는 상황" |
| 30대 | LoL, 디아블로, FC온라인 | "길드 마스터인데 핵심 멤버 두 명이 갈등 중입니다" |
| 40대+ | 바둑/장기 앱, FC온라인 | "온라인 대국에서 AI 치팅 의심 상황" |

#### 프로필 기반 문항 맞춤 예시

같은 역량(협업 리더십)을 측정하는 문항이 프로필에 따라 달라지는 예:

```
20대 남성 LoL   → "4:1 서렌 투표 상황. 당신만 반대. 봇 듀오가 살아날 조짐이 보입니다"
20대 여성 대학생 → "그룹 과제에서 한 명이 아무것도 안 합니다. 발표가 내일인데..."
40대 남성 자영업 → "직원 두 명이 서로 안 맞아서 한 명이 그만두겠다고 합니다"
30대 여성 교사  → "학생이 과제를 AI로 작성해서 제출했습니다"
```

### 게임화 요소

#### 진행 중 재미
- 🔥 **콤보 시스템** — 빠른 연속 응답에 시각적 보상
- ✨ **파티클 이펙트** — 선택할 때마다 이모지 폭발
- 🤖 **비티 캐릭터 가이드** — 문항마다 코멘트 ("흥미로운 선택이네요!")
- 📊 **중간 인터미션** — 절반 지점에서 "지금까지 분석 미리보기"
- 🎵 **배경 분위기 변화** — 문항 진행에 따라 색상 전환

#### UX 언어
- "검사 시작" → **"도전하기"**
- "문항 1/10" → **진행 바 + 스테이지 느낌**
- "제출" → **선택 즉시 자연스럽게 다음으로 전환**
- 모든 문항 텍스트는 **2줄 이내**
- 긴 텍스트 배제, 이모지 + 시각적 선택지 활용

## C. 결과 프레젠테이션

### 검사 대시보드

로그인 후 사용자의 검사 현황을 보여주는 메인 화면:

```
나의 검사 현황
━━━━━━━━━━━━━━━━━━━━━━━
🎭 시나리오 검사    ✅ 완료     5분 전
⚡ 직감 검사        🔓 도전하기  ~4분
🧩 통찰력 검사      🔓 도전하기  ~5분
🤖 AI 활용 검사     🔓 도전하기  ~4분
━━━━━━━━━━━━━━━━━━━━━━━
현재 신뢰도: ⭐ (1/4 완료)
→ 2개 더 하면 ⭐⭐⭐ 정밀 분석 해금!
```

- 검사할 때마다 결과가 누적 업데이트
- 다음 로그인 시 "지난번에 1개 하셨네요, 오늘 1개 더 해볼까요?" 유도
- 4개 완료 시 "풀 프로필 달성!" 축하 + AI 심층 리포트 해금

### 신뢰도 등급 시스템

| 완료 수 | 신뢰도 등급 | 신뢰도 % | 결과 범위 |
|---------|-----------|---------|----------|
| 1개 | ⭐ 탐색 수준 | ~42% | 기본 7차원 결과 |
| 2개 | ⭐⭐ 참고 수준 | ~68% | 교차 검증된 결과 |
| 3개 | ⭐⭐⭐ 신뢰 수준 | ~85% | 정밀 분석 + AI 심층 리포트 해금 |
| 4개 | ⭐⭐⭐⭐ 전문가 수준 | ~95% | 풀 프로필 + 맞춤 전략 |

### 단계적 결과 공개

각 검사 완료 직후, 결과를 단계적으로 드라마틱하게 공개한다:

**1단계: 유형 공개** (즉시)
→ "당신은 ⭐ 통찰형 혁신가입니다" — 흥미 유발 + 공유 욕구

**2단계: 7차원 레이더 차트**
→ 7각형 그래프로 강점/약점 한눈에 파악

**3단계: 통찰력 심층 분석**
→ 이해·분석·예측 3개 하위 점수 펼쳐서 표시

**4단계: 신뢰도 표시 + 다음 검사 유도**
→ "현재 신뢰도 42%. 직감 검사를 추가하면 68%로 올라갑니다"

**5단계: AI 심층 리포트** (3개 이상 완료 시)
→ Claude가 생성하는 개인 맞춤 분석

### 각 검사별 결과 콘텐츠 원칙

**모든 개별 검사 결과에 반드시 포함:**
1. **측정된 차원의 점수 + 해석** — 숫자만이 아니라 "이 점수가 무엇을 의미하는지"
2. **구체적 강점 설명** — "당신은 ~한 상황에서 ~하는 경향이 있습니다"
3. **실생활 적용 팁** — 1개라도 바로 써먹을 수 있는 조언
4. **다른 검사와의 연결 힌트** — "직감 검사를 하면 이 부분이 더 명확해집니다"

**누적 결과에 추가:**
5. **차원 간 조합 분석** — "통찰력은 높은데 AI활용이 낮으면 → 이런 전략"
6. **유형 정밀화** — 검사가 늘수록 유형 분류가 세분화
7. **AI 심층 리포트** — 3개 이상 완료 시 전체를 종합한 맞춤 분석

## 기술 아키텍처

### 재활용 (기존 설계에서)
- **인증**: Supabase Auth (Google OAuth + Kakao OAuth + 이메일 OTP)
- **DB**: Supabase PostgreSQL
- **프레임워크**: Next.js 16 + React 19 + TypeScript + Tailwind CSS
- **AI 분석**: Anthropic Claude API
- **캐싱**: Upstash Redis
- **실시간**: Supabase Realtime (팀/가족 대기 화면)

### 새로 설계 필요
- **문항 데이터 구조**: 7차원 + 테마팩별 시나리오 저장 방식
- **프로필 시스템**: 사용자 프로필 → 테마팩 매핑 로직
- **누적 점수 시스템**: 다중 검사 결과 통합·정규화 알고리즘
- **신뢰도 계산**: 완료 검사 수 + 교차 검증 기반 신뢰도 산출
- **결과 저장**: 검사별 결과 + 누적 결과 DB 스키마

### DB 스키마 변경

기존 `assessment_results` 테이블을 확장:

| 테이블 | 용도 |
|--------|------|
| `user_profiles` | 성별, 연령대, 직업군, 세부 분야 |
| `assessment_results` | 개별 검사 결과 (검사 유형별) |
| `cumulative_results` | 누적 결과 (7차원 통합 점수 + 신뢰도) |
| `assessment_groups` | 팀/가족 그룹 (기존 유지) |
| `assessment_group_members` | 그룹 멤버 (기존 유지) |

### 팀/가족 모드

기존 스펙(`2026-04-15-login-team-family-enhancement-design.md`)의 팀/가족 모드 개념을 유지한다:
- 가족 모드: 관계 개선 방안 + 가족 미션
- 팀 모드: 팀 화합 전략 + 목표 달성 방안
- 초대 링크 기반 그룹 참가
- Supabase Realtime 대기 화면

단, 검사 내용이 7차원·4검사 체계로 바뀌므로, 그룹 분석 프롬프트와 결과 화면은 새로 설계한다.

## 기존 데이터 마이그레이션

기존 6차원 `assessment_results` 데이터는 **보관하되 별도 테이블(`legacy_assessment_results`)로 이동**한다. 6차원→7차원 자동 매핑은 하지 않는다 (차원 정의가 근본적으로 다르므로 매핑이 부정확). 기존 사용자가 로그인하면 "새로운 검사 시스템이 도입되었습니다. 새로 검사해 주세요"로 안내한다.

## 제약 사항

- 테마팩 간 문항의 심리측정 구조(점수 배분)는 반드시 동일해야 한다 (결과 비교 가능성 보장)
- 통찰력 하위 점수 3개는 통찰력 검사(3번)에서만 세분화 측정. 다른 검사에서는 통찰력 종합만 측정.
- AI 심층 리포트는 3개 이상 검사 완료 시에만 생성 (비용 + 데이터 충분성)
- 팀/가족 인원 제한: 2~10명 (기존 유지)
- 1차 출시에서는 테마팩 3~4개로 시작, 점진적 확대
