# BTS 분석 — MVP 제품 전략 문서

> **이 문서의 목적:** BTS 분석 사이트에서 통찰력 검사 1개만으로도 사용자가 "이건 정확하다"라고 느끼고, 심층 해석과 실행 계획을 위해 자연스럽게 결제하게 만드는 구조를 설계한다.

> **범위:** BTS 분석 사이트만 다룬다. 내 직업의 미래 앱, 앱 간 연동, 통합 리포트는 이 문서 범위 밖이다.

---

## 핵심 판단 6가지

1. BTS 분석의 첫 MVP는 **통찰력 검사** 1개다.
2. 무료 결과는 짧지만 **정확해야** 한다. 사용자가 "이거 내 얘기다"라고 느껴야 한다.
3. 첫 유료 상품은 **통찰력 심층 리포트**(12,900원)다.
4. 두 번째 유료 상품은 **90일 통찰력 강화 플랜**(29,000원)이다.
5. 나머지 3개 검사(시나리오/직감/AI활용)는 **Phase 2 확장**이다.
6. 검사 수의 증가는 **신뢰도 상승**으로, 결제는 **해석과 실행**으로 연결된다.

## 섹션 1. BTS 분석 사이트의 본질 정의

### 한 문장 정의

**BTS 분석은 "나는 미래 변화에 얼마나 준비된 사람인가"를 진단하는 미래 준비 상태 검사 서비스다.**

### 왜 일반 성향검사와 다른가

일반 성향검사(MBTI, 에니어그램 등)는 **"나는 어떤 사람인가"**를 분류한다. 분류 자체가 목적이다. 결과를 받으면 "나는 INFP구나"에서 끝난다.

BTS 분석은 다르다. **"나는 앞으로 올 변화에 얼마나 대응할 수 있는가"**를 측정한다. 결과를 받으면 "내 예측력이 약하구나, 여기를 키워야 하는구나"로 이어진다.

차이를 정리하면:

| | 일반 성향검사 | BTS 분석 |
|---|---|---|
| 질문 | 나는 어떤 사람인가? | 나는 미래에 얼마나 준비됐는가? |
| 측정 방식 | 자기 보고 ("나는 ~한 편이다") | 행동 기반 ("이 상황에서 어떻게 하겠는가") |
| 결과 | 유형 분류 | 준비 상태 진단 + 약점 + 실행 방향 |
| 이후 행동 | 없음 (재미로 끝남) | 있음 (훈련·실행·재측정) |

### 왜 미래 준비 상태를 본다고 말할 수 있는가

BTS 분석의 7차원 모델은 WEF(세계경제포럼), OECD, UNESCO 등 5개 국제 기관의 미래역량 프레임워크를 교차 분석하여 도출했다. 이 기관들이 공통적으로 지목하는 역량 — 분석적 사고, 창의성, AI 리터러시, 적응력, 윤리적 책임, 협업 — 이 7차원의 뼈대다.

특히 통찰력(이해-분석-예측)은 OECD의 AAR(Anticipation-Action-Reflection) 사이클에서 "미래를 예측하고 대비하는 능력"으로 정의된 역량을 구조화한 것이다.

즉, BTS 분석이 측정하는 것은 개인의 느낌이나 취향이 아니라, **국제 기관들이 "이것이 미래에 필요하다"고 합의한 역량에 대한 현재 준비 상태**다.

## 섹션 2. 7차원 사용자 언어 재정의

| 차원 | 쉬운 설명 | 이 힘이 약하면 생기는 미래 위험 | 일상에서 보이는 신호 |
|------|----------|-------------------------------|-------------------|
| ⭐ **통찰력** | 복잡한 상황을 꿰뚫어 보고, 앞으로 무슨 일이 벌어질지 읽는 힘 | 변화가 와도 무슨 일이 벌어지는지 늦게 알아차린다. 항상 "몰랐다"는 말을 하게 된다 | 뉴스를 봐도 "그래서 뭐?"가 떠오르지 않는다. 남들이 기회를 잡을 때 뒤늦게 깨닫는다 |
| 🎨 **창의적 혁신** | 정해진 방법 말고 새로운 길을 만들어내는 힘 | 좋은 아이디어가 있어도 실제 변화로 연결하지 못한다. AI가 만드는 것과 차별화가 안 된다 | "원래 이렇게 하는 거야"라는 말을 자주 한다. 새로운 방법을 시도하는 게 불편하다 |
| 🤖 **AI 활용 지능** | AI 도구를 이해하고, 내 일에 효과적으로 쓰는 힘 | AI가 동료가 되는 시대에 혼자 뒤처진다. 도구를 쓰는 사람과 쓰지 못하는 사람의 격차가 벌어진다 | ChatGPT를 써봤지만 "뭘 시켜야 할지" 모른다. AI가 잘못된 답을 줘도 구분하지 못한다 |
| 💎 **감성·소통 지능** | 사람의 마음을 읽고, 신뢰를 만들고, 영향력을 미치는 힘 | 능력이 있어도 사람이 따르지 않는다. AI가 대체하기 가장 어려운 영역에서 경쟁력이 없다 | 회의에서 논리는 맞는데 설득이 안 된다. 팀원이 마음을 열지 않는다 |
| 🔄 **적응 회복력** | 변화가 와도 멈추지 않고, 넘어져도 다시 일어나는 힘 | 변화가 올 때마다 멈추고 무너진다. 한 번의 실패로 오래 멈춰 있다 | 새 시스템이 도입되면 스트레스를 많이 받는다. 계획이 틀어지면 쉽게 포기한다 |
| ⚖️ **윤리적 판단력** | AI 시대에 무엇이 옳은지 판단하고, 책임 있게 결정하는 힘 | 빠른 것만 쫓다가 중요한 것을 놓친다. 기술을 쓸 줄 알지만 써도 되는지 판단하지 못한다 | "효율적이면 됐지"라고 자주 생각한다. 편리함 앞에서 공정성을 잘 따지지 않는다 |
| 🤝 **협업 리더십** | 다양한 사람, 그리고 AI와 함께 목표를 달성하는 힘 | 혼자는 잘하는데 함께 하면 성과가 줄어든다. 사람+AI 혼합 팀에서 역할을 찾지 못한다 | 팀 프로젝트에서 "내가 하는 게 빠르다"고 자주 느낀다. 다른 사람의 강점을 활용하지 못한다 |

## 섹션 3. 4개 검사의 존재 이유 설명

### 왜 4개로 나눴는가 — 사용자 설명

> 사람의 미래 준비 상태를 한 가지 방법으로만 측정하면, 한쪽 면만 보게 됩니다.
> 
> 시나리오를 읽고 판단할 때의 나, 직감으로 즉시 반응할 때의 나, 패턴을 찾고 예측할 때의 나, AI 결과물을 판단할 때의 나 — 이 네 가지 상황에서 드러나는 역량은 다릅니다.
> 
> 각 검사는 다른 각도에서 같은 7차원을 측정합니다. 한 개만 해도 방향이 보이고, 여러 개를 할수록 정확해집니다.

### 검사별 상세

#### 1. 🎭 시나리오 검사

| 항목 | 내용 |
|------|------|
| **주로 측정** | 감성·소통, 윤리적 판단, 협업 리더십, 통찰력(이해) |
| **형식** | 현실적 상황 제시 → 4가지 행동 중 선택 |
| **왜 이 형식이 필요한가** | 실제 상황에서 내리는 판단은 성격이 아니라 역량이다. 어떤 행동을 선택하느냐가 미래 준비 상태를 직접 보여준다 |
| **사용자 설명** | "실제 상황에서 당신이 어떤 판단을 내리는지 봅니다. 머리로 아는 것이 아니라, 상황에서 드러나는 진짜 역량을 측정합니다" |

#### 2. ⚡ 직감 검사

| 항목 | 내용 |
|------|------|
| **주로 측정** | 적응 회복력, 창의적 혁신, 가치 체계(윤리·협업) |
| **형식** | 2개 중 3초 내 직감 선택 + 우선순위 배열 |
| **왜 이 형식이 필요한가** | 3초 안에 선택하면 의식적 포장이 벗겨진다. 꾸며진 답이 아니라 몸에 배인 반응이 나온다 |
| **사용자 설명** | "생각할 틈 없이 선택하세요. 직감이 말하는 것이 당신의 진짜 우선순위입니다" |

#### 3. 🧩 통찰력 검사 ← MVP

| 항목 | 내용 |
|------|------|
| **주로 측정** | 통찰력(이해·분석·예측), 적응 회복력, 창의적 혁신 |
| **형식** | 패턴 발견 + 미래 예측 시나리오 |
| **왜 이 형식이 필요한가** | 통찰력은 BTS 분석의 핵심 축이다. 데이터에서 패턴을 찾고 미래를 예측하는 능력은 자기 보고로 측정할 수 없다. 실제 과제를 풀어야 드러난다 |
| **사용자 설명** | "패턴을 찾고, 흐름을 읽고, 앞으로를 예측하세요. 이 검사가 당신의 통찰 구조를 가장 깊이 보여줍니다" |

#### 4. 🤖 AI 활용 검사

| 항목 | 내용 |
|------|------|
| **주로 측정** | AI 활용 지능, 윤리적 판단, 통찰력(분석) |
| **형식** | AI 결과물 평가 + AI 활용 시나리오 |
| **왜 이 형식이 필요한가** | AI를 "쓸 줄 아는 것"과 "잘 쓰는 것"은 다르다. AI의 오류를 찾고, 한계를 판단하고, 윤리적으로 활용하는 능력은 별도로 측정해야 한다 |
| **사용자 설명** | "AI가 만든 결과를 보고 판단하세요. AI를 도구로 쓸 수 있는 사람인지, 도구에 쓰이는 사람인지 확인합니다" |

## 섹션 4. 사용자 질문 지도

### 단계 1: 기본 자기 이해 질문

| 사용자 질문 | 무료/유료 | 답변 방식 |
|------------|----------|----------|
| "내 통찰력은 어느 정도인가?" | **무료** | 통찰력 종합 점수 + 등급 |
| "이해/분석/예측 중 뭐가 강한가?" | **무료** | 3개 하위 점수 + 간단 한 줄 해석 |
| "나는 어떤 유형인가?" | **무료** | 유형명 + 한 줄 설명 |

### 단계 2: 심화 해석 질문

| 사용자 질문 | 무료/유료 | 답변 방식 |
|------------|----------|----------|
| "왜 이런 결과가 나왔는가?" | **유료 — 심층 리포트** | 통찰 구조 해석 (어떤 판단 패턴이 이 점수를 만들었는지) |
| "내 약점은 구체적으로 뭔가?" | **유료 — 심층 리포트** | 핵심 약점 3개 + 위험 시나리오 |
| "이 결과가 미래에 어떤 영향을 미치나?" | **유료 — 심층 리포트** | 미래 위험 3개 + 대응 방향 |

### 단계 3: 실행 질문

| 사용자 질문 | 무료/유료 | 답변 방식 |
|------------|----------|----------|
| "뭘 먼저 키워야 하나?" | **유료 — 심층 리포트** | 훈련 포인트 3개 + 짧은 행동 제안 |
| "매일 뭘 하면 되나?" | **유료 — 90일 플랜** | 12주 체크리스트 + 주간 루틴 |
| "3개월 후 다시 측정하면 달라질까?" | **유료 — 90일 플랜** | 재측정 포인트 + 변화 추적 |

### 단계 4: 장기 추적 질문

| 사용자 질문 | 무료/유료 | 답변 방식 |
|------------|----------|----------|
| "시간이 지나면 내 역량이 어떻게 변했나?" | **유료 — 월 구독** (Phase 2) | 누적 대시보드 + 변화 그래프 |
| "다른 검사도 해보면 결과가 달라지나?" | **유료 — 추가 검사** (Phase 2) | 다중 검사 교차 분석 |

### 단계 5: 가족/팀 확장 질문

| 사용자 질문 | 무료/유료 | 답변 방식 |
|------------|----------|----------|
| "우리 가족은 어떤 조합인가?" | **유료 — 가족 패키지** (Phase 2) | 가족 역량 시너지 + 관계 미션 |
| "우리 팀의 약한 고리는?" | **유료 — 팀 패키지** (Phase 2) | 팀 역량 프로필 + 역할 제안 |

## 섹션 5. 무료/유료 경계 설계

### 무료에서 보여줄 것 (통찰력 검사 기준)

**원칙: 짧지만 정확해야 한다. 무료만 보고도 "이거 내 얘기 같다"는 느낌이 들어야 한다.**

| 항목 | 내용 | 예시 |
|------|------|------|
| 통찰력 종합 점수 | 0~100 점수 + 등급 | "72점 — 상위 28%" |
| 이해/분석/예측 하위 점수 | 각각 0~100 | 이해 81 / 분석 68 / 예측 55 |
| 통찰 유형 | 유형명 + 한 줄 설명 | "🧠 분석 우위형 — 데이터는 잘 보지만, 흐름은 놓치는 타입" |
| 강점 1줄 | 가장 높은 하위 점수 기반 | "당신은 주어진 정보를 빠르게 파악하는 힘이 강합니다" |
| 약점 1줄 | 가장 낮은 하위 점수 기반 | "그러나 앞으로 어떤 변화가 올지 미리 읽는 힘은 아직 약합니다" |
| 핵심 한마디 | 전체를 관통하는 진단 | "지금은 '이해하는 사람'입니다. '예측하는 사람'이 되면 바뀝니다" |

**무료 결과의 목표:** 사용자가 결과를 보고 "맞아, 나 그래"라고 고개를 끄덕이게 만든다. 그리고 바로 다음 생각이 "그런데 왜 그런 거지? 어떻게 바꾸지?"가 되게 만든다. 이 질문의 답은 유료에 있다.

### 유료에서 잠글 것

| 잠금 항목 | 왜 돈을 낼 가치가 있는가 |
|----------|----------------------|
| **왜 이런 결과가 나왔는지** 구조 해석 | "내가 왜 이런지"를 알아야 바꿀 수 있다 |
| **미래에 생길 수 있는 위험** 3가지 | 위험을 모르면 대비할 수 없다 |
| **핵심 약점** 3가지 상세 분석 | 무엇이 약한지 정확히 알아야 키울 수 있다 |
| **먼저 키워야 할 훈련 포인트** 3가지 | 방향 없이 노력하면 시간만 낭비된다 |
| **짧은 행동 제안** | 오늘부터 시작할 수 있는 구체적 행동 |
| **PDF 저장** | 기록으로 남겨야 실행으로 이어진다 |
| **30일/90일 실행 방향** | 장기적 변화는 계획이 있어야 가능하다 |

### 잠금 표시 방식

무료 결과 화면 하단에 잠긴 항목을 **블러 처리**로 보여준다. 내용의 첫 1~2줄만 살짝 보이고 나머지는 흐리게. 사용자가 "저 안에 내 얘기가 있다"고 느끼게 만든다.

```
┌─────────────────────────────────────┐
│  🔒 당신의 통찰 구조 해석           │
│                                     │
│  당신의 예측력이 낮은 이유는...     │
```
